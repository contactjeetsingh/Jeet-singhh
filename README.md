# Personal Branding Website – Sowrabha GD

A React + Vite powered personal branding website focused on cybersecurity research, insights, and technical blogs.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Markdown-based blog system

## Blog System

Blogs are stored inside:

src/data/insights/

Each blog is written in Markdown with frontmatter metadata.

## Run Locally

npm install
npm run dev
